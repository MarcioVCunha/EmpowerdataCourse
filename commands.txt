Criar um Ambiente =================> python -m venv [nome do ambiente]
Rodar um ambiente =================> executar o arquivo activate dentro da pasta script do seu ambiente
Ver a autorização do terminal =====> Get-ExecutionPolicy
Mudar a autorização do terminal ===> Set-ExecutionPolicy Unrestricted
Criar o arquivo requirements ======> pip freeze > requirements.txt
Instalar todas as libs do projeto => pip install -r requirements.txt
